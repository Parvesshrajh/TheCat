# 🐱 The Cat - Mood-Based Cat Photos

A delightful web application that fetches and displays beautiful cat images based on your current mood! Choose from various moods like happy, sad, cute, playful, and more to see cats that match your vibe.

## ✨ Features

- **Mood-Based Selection**: Choose from 8 different moods (Happy, Sad, Cute, Playful, Sleepy, Crazy, Chill, Love)
- **Random Cat Images**: Fetches fresh cat photos from The Cat API for each mood selection
- **Smooth Animations**: Beautiful loading spinner and fade-in effects
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **Modern UI**: Glassmorphism design with gradient backgrounds and hover effects
- **Error Handling**: Graceful error messages if the API is unavailable

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Click on any mood button that matches how you're feeling
3. Watch the loading animation while we find the perfect cat
4. Enjoy your mood-matched cat photo!
5. Click "Get Another Cat" to see more cats in the same mood

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for the application structure
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **JavaScript (ES6+)**: Fetch API for backend communication
- **The Cat API**: External API for cat image data

## 📦 Setup Instructions

### Option 1: Direct File Opening
1. Clone or download this repository
2. Open `index.html` directly in your web browser
3. That's it! No server required for basic functionality

### Option 2: Local Server (Recommended)
1. Clone or download this repository
2. Navigate to the project directory
3. Start a local HTTP server:
   ```bash
   # Using Python 3
   python3 -m http.server 8000

   # Using Node.js (if available)
   npx http-server

   # Using PHP
   php -S localhost:8000
   ```
4. Open `http://localhost:8000` in your browser

## 🌐 API Information

This application uses [The Cat API](https://thecatapi.com/) to fetch cat images. The API provides:
- High-quality cat photos
- No API key required for basic usage
- Rate limits apply for heavy usage
